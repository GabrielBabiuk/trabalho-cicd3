# Projeto CI/CD - VV&T

Pipeline automatizada com:

- GitHub Actions
- MySQL
- Python
- Pytest

## Funcionalidades

- Geração de códigos sequenciais
- Integração com banco MySQL
- Testes automatizados
- Ambiente 100% automatizado

## Como rodar localmente

1. Instalar dependências:
pip install -r requirements.txt

2. Rodar banco MySQL

3. Executar:
pytest -v
